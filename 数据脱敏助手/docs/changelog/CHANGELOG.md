# 数据脱敏助手 - 更新日志

## v1.1 — 2026-04-08

- 新增：脱敏/还原完成后展示「操作报告」，显示处理了哪些工作表、列、使用了什么策略
- 新增：报告中统计已处理单元格数和跳过的公式单元格数
- 优化：报告以卡片形式嵌入结果页，无需额外文件

## v1.0 — 2026-03-12

- 首个正式版本
- 支持 Excel 文件的自动列类型识别与脱敏
- 支持 8 种脱敏策略：编号替换、随机假名、格式保留替换、等比缩放、等比缩放+扰动、固定偏移、区间映射、日期偏移
- 支持密钥文件导出与数据还原
- 支持基准密钥复用（多次脱敏保持映射一致）
- 支持公式单元格自动跳过
- 支持脱敏前后对比预览
- 完全离线运行，数据不外传
- macOS 原生应用（支持 Apple Silicon 和 Intel）
