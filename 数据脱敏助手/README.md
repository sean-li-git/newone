# 数据脱敏助手

> 完全离线的 Excel 数据脱敏与还原工具，数据不外传

**当前版本**：v1.1  
**最后更新**：2026-04-08

## 功能特点

- 🔒 **完全离线** — 禁止一切网络请求，数据安全有保障
- 📊 **自动识别** — 智能识别列类型（人名、手机号、身份证、金额、日期等）
- 🎯 **8 种策略** — 编号替换、随机假名、格式保留替换、等比缩放、等比缩放+扰动、固定偏移、区间映射、日期偏移
- 🔑 **可逆还原** — 通过密钥文件还原脱敏数据
- 📋 **操作报告** — 脱敏/还原后展示详细的处理报告
- 📐 **公式保留** — 自动跳过含公式的单元格

## 使用方式

### macOS 用户

1. 下载对应架构的 zip 文件：
   - Apple Silicon (M1/M2/M3)：`数据脱敏助手-1.1.0-arm64-mac.zip`
   - Intel：`数据脱敏助手-1.1.0-mac.zip`
2. 解压后得到 `数据脱敏助手.app`
3. **首次打开**：右键点击 app → 选择「打开」→ 在弹窗中再次点击「打开」
4. 之后即可正常双击打开

### 脱敏流程

1. 拖入或选择 Excel 文件
2. 查看自动识别的列类型，按需调整脱敏策略
3. 点击「开始脱敏」
4. 导出脱敏后的 Excel 和密钥文件（密钥请妥善保管！）

### 还原流程

1. 拖入脱敏后的 Excel 文件
2. 拖入对应的密钥文件（JSON）
3. 自动还原并导出
